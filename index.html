Что такое метод мат. индукции и зачем он нужен?

Метод полного перебора конечного числа случаев, исчерпывающих все возможности, называется полной индукцией. Этот метод имеет крайне ограниченную область применения в математике, так как обычно математические утверждения касаются бесконечного множества объектов (например, натуральных чисел, простых чисел, квадратов и т.п.) и перебрать их невозможно.

Существует метод рассуждений, который позволяет заменить неосуществимый бесконечный перебор доказательством того, что если утверждение истинно в одном случае, то оно окажется истинным и в следущем за ним случае. Этот метод носит название метод математической индукции (или рассуждением от n к n+1).

На чем основан метод мат. индукции?

В основе метода математической индукции (ММИ) лежит принцип математической индукции:

Принцип математической индукции
Утверждение P(n) (где n - натуральное число) справедливо при ∀n ∈ N, если:
1. Утверждение P(n) справедливо при n=1.
2. Для ∀k ∈ N из справедливости P(k) следует справедливость P(k + 1).
Как работать с этим методом?

Доказательство с помощью метода математической индукции проводится в два этапа: база и шаг индукции.

Доказательство с помощью метода мат. индукции
1. База индукции Проверяется истинность утверждения при n=1 (или любом другом подходящем значении n)
2. Индуктивный переход (шаг индукции) Считая, что справедливо утверждение P(k) при n = k, проверяется истинность утверждения P(k + 1) при n = k + 1.
